## Overview

MatchMind AI is an AI-powered football analysis platform designed to help
fans understand the story behind a match. Instead of only displaying
statistics, MatchMind AI explains momentum shifts, tactical changes,
substitutions, and key match events using human-friendly insights generated
by IBM Granite.

The platform transforms complex football data into clear explanations that
improve fan understanding, engagement, and trust.

---

## Problem Statement

Football fans can easily access match statistics such as possession, goals,
cards, and substitutions. However, these numbers often fail to explain:

- Why momentum shifted during a match
- Why a tactical change succeeded or failed
- Why a substitution changed the outcome
- Which moments truly influenced the game

This creates a gap between match data and fan understanding.

---

## Solution

MatchMind AI uses Explainable AI to analyze football match events and
generate easy-to-understand explanations.

The platform:
- Detects momentum shifts
- Identifies key turning points
- Explains tactical changes
- Analyzes substitutions and cards
- Generates insights for different audience levels

By combining football analytics with AI-generated explanations, MatchMind AI
helps users understand not only **what** happened, but **why** it happened.

---

## Key Features

###  Momentum Shift Detection
Identifies periods where control of the match changes significantly.

###  Explain This Moment
Provides AI-generated explanations for goals, substitutions, yellow cards,
and red cards.

###  Tactical Analysis
Explains tactical adjustments and their impact on match outcomes.

###  Multi-Language Support
Generates explanations in:
- English
- Hindi
- Telugu

###  Fan Knowledge Levels
- **Beginner Mode** — simple, jargon-free explanations
- **Fan Mode** — moderate depth with context
- **Analyst Mode** — detailed tactical and statistical breakdown
